Elden Ring Team Creator

This app allows users to create a team of 3 Elden Ring Classes. Each character has stats, a random weapon, shield, and 3 items and can each be renamed and re-rolled. This project was created using React.js, React Bootstrap, all 4 CRUD operations, and React Route.

Top 40 Music App

Converted the app below to use the appwrite (https://appwrite.io/) backend/database to hold the user data. The app uses HTML, CSS, and JavaScript, and allows a user to input an artist, song, and year, which then gets put into the song list (as long as the list is under 40 songs) and then allows the user to upvote/downvote existings songs on the list. The songs are ordered in real time as the user votes! Songs are saved in the appwrite database. Page is responsive to all devices!

Top 40 Music App

This app allows for a Top 40 list for current artists/songs. Users input an artist and song, the song then gets added to the list of top songs, and users can then upvote and downvote songs on the list! This app was created using EJS, CSS, JavaScript, Node.js/Express.js, and utilizes all four CRUD operations. MongoDB was used for the database, and this project is hosted on onRender.com

Game of War App

Cocktail Recipe App

This app allows users to select a basic type of alcohol, which then pulls a random cocktail recipe (along with cocktail image) of the chosen alcohol from a Cocktail API. The app then carousels through different cocktail recipes for the chosen alcohol until the user stops on a recipe they like.
